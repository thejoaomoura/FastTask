Nesta pasta devem ser adicionados os ícones utilizados pela aplicação.

Ícones recomendados (a serem criados ou baixados):

1. app_icon.png - Ícone principal do aplicativo
2. generic.png - Ícone padrão para processos
3. browser.png - Ícone para navegadores
4. folder.png - Ícone para processos de gerenciamento de arquivos
5. terminal.png - Ícone para terminais/consoles
6. text.png - Ícone para editores de texto
7. document.png - Ícone para processadores de texto
8. spreadsheet.png - Ícone para planilhas
9. code.png - Ícone para IDEs/editores de código

Tamanho recomendado: 32x32 pixels ou maior (com boa resolução).
Formato recomendado: PNG com transparência.

Você pode criar seus próprios ícones ou obter ícones gratuitos em:
- https://www.flaticon.com/
- https://icons8.com/
- https://www.iconfinder.com/

Certifique-se de verificar as licenças de uso antes de utilizar ícones de terceiros.
